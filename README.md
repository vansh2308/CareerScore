
# Career Score
#### Built With NextJs, Typescript, Supabase, FramerMotion

___

<img src="/public/hero.png" width="100%" style="border-radius:10px">


### Features
- Intriguing Dark UI
- Magic Link Auth
- Role Based Access Control 
- Interactive User Dashboard
- Realtime state management 
- Resume Bucket
- Drag-n-drop Resume uploader
- Resume previewer
- Resume Scorer, Analytics 
- Resume Comments, Messaging System


### Setup Instructions
Run the following in terminal
```bash
git clone https://github.com/vansh2308/CareerScore.git
cd ./CareerScore
npm install
```
Setup your .env file 
```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
JWT_SECRET=d4
GOOGLE_APP_PASSWORD=
NEXTAUTH_URL=http://localhost:3000
NEXT_PUBLIC_SITE_URL=
```

Open terminal and issue:
```
npm run dev
```

Voila ;) Just type in http://localhost:3000 in your browser!
